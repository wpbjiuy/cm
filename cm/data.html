首页

使用获取\设置数据请加载data.js
FnData(dataName/*获取key值的元素属性名称*/, cut/*下一级对象的拆分点*/) 为构造函数，fnData = new FnData('data-name', '-');

使用数据绑定请加载dataBind.js

获取数据

使用方法：fndata.getData(dom,isEmpty)

	dom => 数据节点层
	isEmpty => 是否返回空数据
实例： 
	DOM: 
	<div id="fnData">
		<div><input type="text" data-name="name"/></div>
		<div data-name="has" data-type="array">
			<input type="text" />
			<input type="text" />
		</div>
		<div data-name="tms" data-type="array" data-arType="objAr">
			<div data-ar="obj">
				<input type="text" data-arname="a" />
				<input type="text" data-arname="b" />
			</div>
			<div data-ar="obj">
				<input type="text" data-arname="a" />
				<input type="text" data-arname="b" />
			</div>
		</div>
		<div data-name="objz" data-type="object">
			<input type="text" data-name-1="fd" />
			<div data-type="array" data-name-1="ar">
				<input type="text" />
				<input type="text"/>
			</div>
		</div>
	</div>

	数据：{
		name:'String',
		has:['String','String'],
		tms:[
			{a:1,b:2},
			{a:11,b:22}
		],
		objz:{
			fd:'String',
			ar:['String','String']
		}
	}

数据赋值

使用方法：fndata.setData(dom, data)

	dom => 数据节点层
	data => 数据

数据绑定

使用方法：dataBind.run(s); dataBind.atKeyVal(mt,t)

设置元素属性 data-bind="dataName"

	s => 绑定集合 [dataName,dataName,...]
	mt => 绑定的数据名 dataName
	t => 初始值（如没哟，则获取输入框的值） String
